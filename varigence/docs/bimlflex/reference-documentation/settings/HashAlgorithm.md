# Hash Algorithm

The [hashing](bimlflex-concepts-hashing) algorithm to use. ("MD5"/"SHA1"/"SHA2_256"/"SHA2_512").

Notes:

* This setting is part of the *Core* settings category.
* The default value for this setting is `SHA1`.

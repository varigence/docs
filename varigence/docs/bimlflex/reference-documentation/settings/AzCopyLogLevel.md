# Log Level

The log level for AzCopy v.10 logs. Available log levels are: "NONE", "DEBUG", "INFO", "WARNING", "ERROR", "PANIC", and "FATAL"

Notes:

* This setting is part of the *AzCopy* settings category.
* The default value for this setting is `NONE`.

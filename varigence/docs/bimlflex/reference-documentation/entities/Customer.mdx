# Customers

The BimlFlex [**Customer**](../../concepts/customer) offers a flexible way to configure different functional areas which act as top-level groupings of metadata and models.

## Overview
  
| <div style={{ width: '200px' }}>Property</div> | Description |
| --------- | ----------- |
|Name | The name of the Customer (tenant). This is a unique name within the BimlFlex database.|
|Active | Indicator whether the Customer is set as Active (enabled).|
